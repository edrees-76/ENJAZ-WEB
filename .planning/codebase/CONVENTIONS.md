# Coding Conventions
- **Backend (C#):** Standard ASP.NET Core conventions. Usage of Dependency Injection for Service layers. Endpoints use standard `[ApiController]` decorators.
- **Frontend (TS/React):** Functional components with React Hooks. PascalCase for component files, camelCase for methods.
- **Styling:** Adhere strictly to the "Glassmorphism" minimal UI design language established by the Enterprise Architecture framework.
