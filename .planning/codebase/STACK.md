# Stack & Technology
- **Backend:** ASP.NET Core 8 Web API
- **Database:** SQLite (`enjaz.db`) managed via Entity Framework Core (Migrations)
- **Frontend:** React 18 with TypeScript, packaged by Vite
- **UI:** Custom Glassmorphism / Minimalist interface
- **Architecture:** Local-first, client-server split

*Note: The project was migrated from a legacy WPF desktop app to this modern tech stack.*
